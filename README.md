# Java Spring - Maven Project
  Aplicação que demonstra o uso do framework Spring MVC, Java e Maven, criando uma conexão com banco de dados local Oracle.
 
## Ambiente de Desenvolvimento
- Eclipse Neon
- Java 1.8
- Maven 3.3.9
- Spring 5.1.RELEASE
- Tomcat 7
