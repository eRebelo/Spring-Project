# Java Spring - Maven Project
  Aplicação Java que demonstra o uso do framework Spring MVC utlizando Hibernate + JPA para persistência de dados no banco.
 
## Ambiente de Desenvolvimento
- Eclipse Neon
- Java 1.8
- Maven 3.3.9
- Spring 5.1.1
- Hibernate 5.3.7
- JPA 2.1
- Tomcat 7
- Oracle Database
